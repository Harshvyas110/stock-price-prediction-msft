# stock-price-prediction-msft
MSFT Stock Price Direction Prediction using Machine Learning (XGBoost)  This project predicts whether Microsoft (MSFT) stock will go UP or DOWN tomorrow, using:  ✅ Technical indicators ✅ Feature engineering ✅ Machine learning (XGBoost Classifier) ✅ 15+ years of stock data ✅ Accuracy up to 47–50% (very normal for real markets)
